# Struktur Project Web FFT

Dokumen ini menjelaskan struktur folder utama project Web FFT.

## backend

Berisi aplikasi Flask, route API, template admin, file statis backend, upload konten, dan database aktif.

## frontend

Berisi halaman publik website, stylesheet, script, aset gambar, ikon, dokumen, dan halaman akademik atau mahasiswa.

## database

Berisi file pendukung database seperti backup SQL lokal. Database aktif tidak dipindahkan agar sistem yang berjalan tidak rusak.

## docs

Berisi dokumentasi teknis, catatan desain, alur backend, dan daftar fitur.

## scripts

Berisi script utilitas untuk pemeriksaan dan cleanup project.

## .local-backups

Berisi backup lokal, arsip zip, dan hasil backup sementara. Folder ini tidak ikut Git karena hanya dipakai untuk keamanan lokal.
