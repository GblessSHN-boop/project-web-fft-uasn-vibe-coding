PANEL ADMINISTRASI DATA

Silakan pilih kategori data yang ingin Anda kelola melalui panel administrasi. Halaman ini digunakan untuk mengakses pengelolaan informasi dekan dan data dosen secara terpisah agar proses administrasi lebih tertib, jelas, dan terstruktur.

DK

DATA DEKAN

Kelola informasi resmi dekan, termasuk identitas, status kepegawaian, foto formal, dan file biodata yang digunakan untuk kebutuhan administrasi fakultas.

MASUK KE DATA DEKAN
DS

DATA DOSEN

Kelola seluruh data dosen, termasuk identitas, foto utama, thumbnail frontend, serta file biodata yang digunakan dalam sistem administrasi dan publikasi internal.

MASUK KE DATA DOSEN
BR

DATA BERITA

Masuk ke modul berita untuk memilih tipe konten yang ingin dikelola, yaitu Kelola Berita atau Kelola Banner Iklan secara terpisah.

MASUK KE DATA BERITA
Gunakan menu di atas untuk masuk ke modul pengelolaan data yang sesuai.
