# Web FFT UASN

Website Fakultas Filsafat Teologi Universitas Advent Surya Nusantara.

## Struktur Utama

- `frontend/pages`: halaman HTML utama.
- `frontend/css`: file styling frontend.
- `frontend/js`: file interaksi frontend.
- `frontend/assets`: gambar, ikon, video, dokumen, dan file publik frontend.
- `backend`: aplikasi Flask, template admin, static admin, dan data backend.
- `database`: file database, migrasi, dan backup database.
- `docs`: dokumentasi teknis project.
- `public`: file publik untuk deployment.
- `scripts`: script bantuan project.

## Catatan

Struktur ini dibuat untuk memisahkan frontend, backend, database, dokumentasi, dan aset agar project lebih mudah dirawat.
